# Konfiguration

Die Konfiguration ist nach Modulen gegliedert.

- `ma_variants/`: Beispielkonfigurationen fuer Parameter, Optionen, Systeme, Naming, Wirtschaftlichkeit, IDA-Export, Datenbank und Kataloge.
- `ma_analyse/`: vorbereiteter Bereich fuer spaetere Analysekonfigurationen.
- `ma_weather/`: vorbereiteter Bereich fuer Wetterdatenanalyse.
- `common/`: vorbereiteter Bereich fuer uebergreifende Konfigurationen.
