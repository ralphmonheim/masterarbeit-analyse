# ma_assessment

- **Zweck:** Technische, wirtschaftliche und oekologische Ergebnisse aggregieren und bewerten.
- **Eingaben:** Ergebnisse aus `ma_analyse`, `ma_economy` und
  `ma_sustainability`; spaeter Mengen-, Bauteil- und Oeffnungsinformationen
  aus freigegebenen `ma_building`-Modellversionen.
- **Ausgaben:** Scores, Rankings, Pareto-Loesungen und Entscheidungsvorlagen.
- **Abgrenzung:** keine primaere Fachberechnung und keine Berichtserzeugung.
- **Abhaengigkeiten:** `ma_analyse`, `ma_economy`, `ma_sustainability`; Phase 5.
- **Status:** geplant.
- **Naechster Schritt:** Bewertungsregeln und Umgang mit Scoring/Pareto festlegen.
