# Entwuerfe: Anfragen an VDE VERLAG sowie VDI/DIN Media

Stand: 2026-07-14
Status: nicht versendet; technische und vertragliche Vorpruefung, keine Rechtsberatung

Die Hochschulbibliothek bestaetigt den Umfang ihrer Vertraege. Diese Entwuerfe
fragen davon getrennt nach Rechten, die nur der jeweilige Rechteinhaber oder
Vertragspartner einraeumen kann.

## VDE VERLAG

Betreff: Individuelle Erlaubnis fuer begrenzte Verarbeitung von VDE-Regelwerken in einer Masterarbeit

Sehr geehrte Damen und Herren,

im Rahmen einer nichtkommerziellen Masterarbeit entwickle ich eine lokale
Python-Software fuer nachvollziehbare TGA-Berechnungen. Die aktuellen
VDE-VERLAG-AGB und der oeffentliche NormenBibliothek-Nutzungsvertrag 2026
untersagen beziehungsweise beschraenken KI- und TDM-Verarbeitung. Deshalb
bitte ich vor jeder Verarbeitung um eine schriftliche Einzelfallauskunft:

1. Kann fuer ausgewaehlte DIN-VDE-Normen oder VDE-Regelwerke eine individuelle
   Erlaubnis fuer eine isolierte lokale KI-Umgebung erteilt werden?
2. Waeren ChatGPT Enterprise, Edu, Codex oder eine API ohne Training umfasst?
3. Duerfen einzelne technische Anforderungen, Formeln oder Grenzwerte in
   begrenztem Umfang manuell oder maschinell strukturiert werden?
4. Darf daraus eine eigenstaendige maschinenlesbare fachliche Abstraktion
   ohne Normersatzcharakter entstehen?
5. Welche Nutzer-, Speicher-, Loesch-, Sicherheits- und
   Veroeffentlichungsbedingungen waeren einzuhalten?

Ohne passenden schriftlichen Rechtebeleg werden keine geschuetzten Inhalte
mit KI oder Automatisierung verarbeitet.

## VDI als Rechteinhaber

Betreff: Rechte fuer Software, Auszuege und begrenzte Verarbeitung von VDI-Richtlinien

Sehr geehrte Damen und Herren,

im Rahmen derselben Masterarbeit moechte ich als Rechteinhaberauskunft
klaeren, ob fuer einzelne VDI-Richtlinien eine projektspezifische
Nutzungserlaubnis erhaeltlich ist. Der VDI nennt Softwareverwertung,
Abdruck und elektronische Vervielfaeltigung als lizenzpflichtige Nutzungen.

Bitte teilen Sie mir schriftlich mit:

1. Ist eine individuelle Erlaubnis fuer KI-Verarbeitung, begrenzte
   Anforderungsextraktion oder fachliche Strukturierung erhaeltlich?
2. Darf eine isolierte lokale KI ohne Training verwendet werden?
3. Duerfen eigene Paraphrasen, Parameter und Implementierungsregeln in einem
   privaten Repository gespeichert und spaeter ohne geschuetzte Inhalte
   veroeffentlicht werden?
4. Welche Bedingungen gelten fuer Zitate, Formeln, Tabellenfragmente,
   Abbildungen und die Implementierung von Software auf Grundlage einer
   Richtlinie?

## DIN Media als Vertrieb und Vertragspartner

Betreff: Nautos-, Netzwerk- und Verarbeitungslizenz fuer VDI-Richtlinien in einer Masterarbeit

Sehr geehrte Damen und Herren,

DIN Media weist derzeit kein mit der DIN-KI-Lizenz vergleichbares
Standardangebot fuer VDI aus. Bitte teilen Sie mir fuer die konkret benannten
VDI-Dokumente schriftlich mit:

1. Welche Produkt-, Download-, Netzwerk- oder Mehrplatzbedingungen gelten bei
   Bezug ueber DIN Media beziehungsweise Nautos?
2. Deckt die Frankfurt-UAS-Hochschullizenz irgendeinen maschinellen oder
   KI-gestuetzten Verarbeitungsschritt ab?
3. Ist eine individuelle Verarbeitungslizenz erhaeltlich, und welche Systeme,
   Nutzer, Speicherorte und Veroeffentlichungsformen waeren davon umfasst?
4. Welche zusaetzliche Zustimmung des VDI als Rechteinhaber ist erforderlich?

## Noch einzusetzen

- konkrete Dokumentnummern und Ausgaben
- vorgesehener Verarbeitungsumfang
- eingesetztes System und Betriebsmodell
- Speicherort, Schutzmassnahmen und Loeschfrist
- geplanter Umfang der Masterarbeit und einer moeglichen Veroeffentlichung
- Name, Studiengang und Kontakt
