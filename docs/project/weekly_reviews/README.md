# Weekly Reviews

Dieser Ordner sammelt Wochenzusammenfassungen fuer das Masterarbeitsprojekt.

## Zweck

Die Wochenberichte dokumentieren, was in einer Woche fachlich, technisch und
organisatorisch passiert ist. Sie ersetzen nicht `PLAN_STATUS.md`, sondern
ergaenzen ihn als Archiv.

## Ausloeser

Ein Wochenbericht wird mit der Codex-Routine `wochenabschluss` oder der
Ausloesephrase `Eine schoene Woche.` vorbereitet.

## Inhalt

Ein Wochenbericht soll knapp festhalten:

- abgeschlossene Arbeiten
- geaenderte Module
- getroffene Nutzerentscheidungen
- technische Entscheidungen
- Releases oder Tags der Woche
- offene Aufgaben nach Modulen
- Risiken oder Blocker
- empfohlene naechste Wochenprioritaeten

## Benennung

Neue Wochenberichte sollen nach Kalenderwoche benannt werden, zum Beispiel:

```text
2026-KW24.md
```

## Archivregel

Wochenberichte bleiben in diesem Ordner. Umgesetzte Plaene werden weiterhin
unter `docs/project/archive/plans/` archiviert.
