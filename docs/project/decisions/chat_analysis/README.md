# Chat-Analyse

Dieser Bereich ist fuer spaetere Analysen exportierter Chats vorbereitet.

Codex kann vergangene ChatGPT-Chats nur auswerten, wenn sie als Text, Markdown, PDF oder andere lesbare Dateien im Projektordner vorhanden sind.

Regeln:

- Nur masterarbeitsbezogene Chats auswerten.
- Website-Themen ausschliessen.
- Nur echte Nutzerentscheidungen extrahieren.
- Keine Entscheidungen erfinden.
