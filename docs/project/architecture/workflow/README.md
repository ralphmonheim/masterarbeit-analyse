# Workflow-Diagramme

Dieser Ordner enthaelt Workflow-Diagramme und zugehoerige Strukturreviews.

## Ablageregel

- Originalgrafiken werden unveraendert abgelegt.
- Neue Fassungen erhalten eine eigene Versionsnummer.
- Nur die aktuelle Fassung und ihr Review bleiben in diesem Ordner.
- Ersetzte Fassungen werden nach `docs/project/archive/workflow/` verschoben.
- Eine Grafik wird nicht allein durch die Ablage zur verbindlichen
  Zielarchitektur.
- Fachliche Aenderungen werden zuerst im Review dokumentiert und danach mit
  `TARGET_ARCHITECTURE.md` sowie dem Masterarbeitsleitfaden abgeglichen.

Vorgesehenes Dateiformat:

```text
WORKFLOW_DIAGRAM_vX.Y.Z_YYYY-MM-DD.<format>
WORKFLOW_DIAGRAM_REVIEW_vX.Y.Z_YYYY-MM-DD.md
```
