# Archivierte Plaene

Dieser Ordner enthaelt umgesetzte, ersetzte oder historische Planstaende.

Aktive Planung:

- `docs/project/plans/PLAN_STATUS.md`
- `docs/project/plans/PLAN_INDEX.md`
- `docs/project/plans/inbox/`

Regel:

- Neue Plaene kommen zuerst in `docs/project/plans/inbox/`.
- Umgesetzte oder ueberholte Plaene werden nach `docs/project/archive/plans/`
  verschoben.
- Nach dem Verschieben werden `PLAN_INDEX.md` und `PLAN_STATUS.md`
  aktualisiert.
