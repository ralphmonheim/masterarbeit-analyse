# Projektarchiv

Dieser Ordner enthaelt alte, ersetzte oder externe Referenzdokumente. Das
Archiv ist nicht die aktive Projektsteuerung.

Aktive Steuerdateien bleiben weiterhin unter:

- `docs/project/plans/`
- `docs/project/decisions/`
- `docs/project/architecture/`
- `docs/project/MASTERARBEIT_LEITFADEN.md`

Archivbereiche:

| Ordner | Inhalt |
|---|---|
| `leitfaeden/` | alte oder externe Leitfadenfassungen |
| `plans/` | umgesetzte, ersetzte oder historische Planstaende |
| `workflow/` | ersetzte Workflow-Grafiken und zugehoerige Reviews |

Archivierte Dokumente werden nicht geloescht, sondern nur als Referenz
aufbewahrt.
